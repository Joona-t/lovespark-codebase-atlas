The Data

Inputs

  • CONFIG object (canvas size, grid density, scene name, palette, sampling strategy)
  • p5.js draw loop frame callback at 30 FPS
  • Scene function returns offscreen p5.Graphics with rendered geometry
  • Recording mode: frame number → capture PNG

Outputs

  • Visible canvas with character-grid composited on top of geometric source
  • Optional ZIP of PNG frames (glyph-grid_*.zip) when CONFIG.record=true
  • FFmpeg-compatible frame sequence for GIF/video export

The Naive Version

Draw a sine wave into an offscreen buffer. Walk a 10×10 grid, print . for dark cells and # for bright. Done.

The Core Loop

File: index.html lines 200-400 (the p5.draw function + glyph sampling loop)

Role: Walk an N×M grid over source Graphics buffer, sample brightness/color per cell, pick character from ramp, render at source hue. · detection: audit-meta.yml override

The Verification Loop

No formal tests; verification is visual + byte-identity under CONFIG.compat='v1'. Eight frozen pieces must regenerate byte-identically. CR-5 caught under-sampled Gaussian in XDoG. No runtime errors in recording mode (deterministic seed).

The Assumptions

  • Browser has p5.js 1.11.2+ from CDN (no offline mode)
  • User will edit CONFIG in the HTML source and reload (no UI controls)
  • Recording mode uses JSZip and browser download API (not Node.js)
  • System has ffmpeg on PATH for export-gif.sh (optional; skipped if not present)

The Decisions

DateDecision
2026-04-16Implemented brute-force shape-atlas lookup before k-d tree optimization
At n≈256 shapes, k=6 dims: tight Float32Array loop often equals or beats pointer-chasing k-d tree in V8. Profile first, tree second.
2026-04-16Rec.709 weights for linearized luminance, not Rec.601
Rec.601 weights are gamma-encoded; Rec.709 applies to linear-light sRGB after EOTF. Category error caught in code review.
2026-04-16Bundled Cascadia Mono WOFF2 subset for Unicode 16 octants fallback
No mainstream OS ships octants by default (Sep 2024). Runtime cascade: octant → sextant → braille → block covers system variance.

The Trade-offs

  • No GPU rendering path initially; error-diffusion dither cannot run on WebGL (no inter-fragment communication). Validator auto-downgrades to Bayer.
  • Single HTML file means no module system; all CONFIG at top, no layering
  • Recording mode is deterministic but unoptimized; 4sec × 30fps on high grid density can be slow
